Creating Content Using JDBC and SQL

We will interact with relational databases using JDBC and the Structured Query Language (SQL). SQL is a domain specific language designed for updating and retrieving data in table-based databases. We will not cover it in class but you will likely need to use it. SQL is very easy and descriptive. In most cases, a simple Google search will be sufficient to find "how to" do something with SQL. My go-to place to lookup SQL syntax is https://www.w3schools.com/sql/.

tip

If you are interested in learning more, SQLZoo is one of the best and popular website for learning SQL online. For general knowledge, checkout SQL on Wikipedia.

Recall, from previous section, that we have created a connection to our SQLite database through JDBC:

final String URI = "jdbc:sqlite:./MyBookApps.db";
Connection conn = DriverManager.getConnection(URI);

Create a table#

To create a table, we can execute the following SQL statement:

"CREATE TABLE IF NOT EXISTS Books (title VARCHAR(100), isbn VARCHAR(14),
publisher VARCHAR(14), year INTEGER, author VARCHAR(100));

Notice SQL reads like English! There is really no need to explain it; is there?!

We can execute a SQL statement by (1) creating a Statement object and (2) invoking its execute method, passing the SQL statement as an argument to it.

Statement st = conn.createStatement();
String sql = "CREATE TABLE IF NOT EXISTS Books (title VARCHAR(150), isbn VARCHAR(14),
publisher VARCHAR(150), year INTEGER, author VARCHAR(100));";
st.execute(sql);

Insert data into table#

To insert data, we can execute a SQL statement such as:

INSERT INTO Book(title, isbn, publisher, year, author)
VALUES ('1984', '978-0451524935', 'Signet Classic', 1961, 'George Orwell');

We can reuse the Statement object (st) to execute the aforementioned INSERT statement.

sql = "INSERT INTO Book(title, isbn, publisher, year, author)
VALUES ('1984', '978-0451524935', 'Signet Classic',
1961, 'George Orwell');";
st.execute(sql);

Let's insert two more!

sql = "INSERT INTO Book(title, isbn, publisher, year, author)
VALUES ('The Glass Hotel', '978-0525521143', 'Knopf',
2020, 'Emily St. John Mandel');";
st.execute(sql);
sql = "INSERT INTO Book(title, isbn, publisher, year, author)
VALUES ('The Catcher in the Rye', '978-7543321724',
'Little, Brown and Company', 1991, 'J.D Salinger');";
st.execute(sql);

Search for records in a table#

SQL is particularly powerful when it comes to querying data. Here is a simple example to search for all courses where the course name contains he characters:

SELECT * FROM Book
WHERE title LIKE '%he%';

Let's run this query using JDBC:

sql = "SELECT * FROM Books WHERE title LIKE '%he%';";
ResultSet rs = st.executeQuery(sql);
while (rs.next()) {
System.out.println("Book title: " + rs.getString("title"));
System.out.println("Book author: " + rs.getAuthor("author"));
}

Notice I've used a different method, executeQuery to run the above SQL statement. The executeQuery method returns a ResultSet object which you can iterate over to extract the retrieved data.

Parameterized SQL with JDBC PreparedStatement#

A JDBC PreparedStatement is a special kind of JDBC Statement object with some useful additional features. Namely, it makes it easy to insert parameters into the SQL statement.

Book b1 = new Book ("The Catcher in the Rye", "978-7543321724",
"Little, Brown and Company", 1991, 1);
PreparedStatement sql = "INSERT INTO Books (title, isbn, publisher, year, author)
VALUES (?, ?, ?, ?, ?);";
pst = conn.prepareStatement(sql);
pst.setString(1, b1.getTitle());
pst.setString(2, b1.getIsbn());
pst.setString(3, b1.getPublisher());
pst.setInt(4, b1.getYear());
pst.setString(5, b1.getAuthor());
pst.execute();
